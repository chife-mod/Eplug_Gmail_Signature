Install in Gmail

  1. Open the raw file signature-v3.html in Chrome (button above).
  2. Select all — Cmd+A — then copy — Cmd+C.
  3. Gmail → ⚙️ → See all settings → General → Signature.
  4. Paste — Cmd+V — into the signature field.
  5. Scroll down and hit Save Changes.

📋 How to Install in Gmail

  1. Open signature.html (or signature-live.html) in Chrome
  2. Select all: Cmd+A (Mac) / Ctrl+A (Windows)
  3. Copy: Cmd+C / Ctrl+C
  4. Open Gmail → ⚙️ Settings → See all settings → General → Signature
  5. Paste: Cmd+V / Ctrl+V
  6. Click Save Changes at the bottom
